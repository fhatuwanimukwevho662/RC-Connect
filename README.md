# RC Connect

RC Connect is an academic support platform designed for Rosebank College students. 

The system allows students to:
- Upload and share study notes
- Join study groups
- Track assignments and deadlines
- Participate in discussion forums
- Access peer tutoring services

This prototype was developed using HTML, CSS, and JavaScript for the HCIN6221 POE project.
